ES6开讲
========

## 概要

ECMAScript2015(ES6)是Javascript最标准的语法式样，是在2015年6月由Ecma国籍组织公布的最新版本，现在已经被多个领域和浏览器所广泛采纳和使用。

## 新特性

* class类定义
* import/export模块化
* 函数定义方法改进（默认值，可变参数，箭头函数）
* let/const作用域定义
* for...of列举
* Promise,Map/Set/Proxy嵌入式对象追加
* etc.
